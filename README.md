### 🌠 The Odyssey
**Stellar-drift** is a mesmerizing, browser-based cosmic survival experiment. Navigate an ethereal living galaxy, command automated weaponry, and unleash celestial abilities to survive endless waves of stellar adversaries. 

Crafted with **HTML5 Canvas**, this project offers a seamless blend of arcade action and elegant, atmospheric design. ☄️

---

## ✨ Features

* 🛰️ **Dynamic Combat:** Automated weaponry systems that let you focus on movement and strategy.
* 🔮 **Celestial Abilities:** Unleash powerful skills to clear the screen of adversaries.
* 🧪 **Procedural Visuals:** A "living galaxy" feel powered by high-performance Canvas rendering.
* ⚛️ **Modern Tech Stack:** Built with React 19 and TypeScript for a robust, type-safe experience.
* 📱 **Mobile Ready:** Integrated with Capacitor for potential cross-platform exploration.

---

## 🛠️ Tech Stack

* **Framework:** React
* **Build Tool:** Vite
* **Language:** TypeScript
* **Graphics:** HTML5 Canvas API
* **Styling:** CSS3 (Custom atmospheric effects)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
Make sure you have **Node.js** installed on your machine.

### Installation

1. Clone the repo:
   git clone https://github.com/brunda-coder/Stellar-drift.git

2. Install NPM packages:
   npm install

3. Run the development server:
   npm run dev

---

## 🎮 How to Play

* **Move:** Use your mouse or touch to navigate the ship through the cosmic void.
* **Survive:** Avoid colliding with stellar adversaries while your automated turrets handle the offense.
* **Dominate:** Collect power-ups and use abilities to climb the leaderboard.
